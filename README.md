## 📸 About
 
This is the official website for **Sri Sai Furniture** — a premium furniture studio based in Chhatrapati Sambhajinagar (Aurangabad), Maharashtra. The website showcases all furniture services, contact details, team information, and showroom location.
 
---
 
## ✨ Features
 
- ⚡ **Pure HTML/CSS/JS** — no frameworks, loads instantly
- 🎨 **Warm light theme** — ivory, walnut & sage color palette
- 📱 **Fully responsive** — works on mobile, tablet & desktop
- 🧭 **Sticky navbar** with smooth scroll & mobile hamburger menu
- 🎞️ **Scroll animations** — elements reveal as you scroll
- 🏷️ **Live scrolling ticker** — highlights all services
- 📍 **Animated location pin** with Google Maps link
- 💬 **Floating WhatsApp & Call buttons** always visible
- 🔤 **Premium typography** — Fraunces serif + Outfit sans-serif

## 🚀 Deployment (GitHub Pages)
 
1. Upload `index.html` to this repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Click **Save** — site goes live in ~1 minute at:
   `https://YOUR-USERNAME.github.io/sri-sai-furniture/`
